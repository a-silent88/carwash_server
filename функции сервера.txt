1) StartSession
Нужно вызывать в начале работы клиента с терминалом
В URL передается terminalId - какой-нибудь уникальный идентификатор терминала, чтобы потом понимать где сколько денег
в ответ приходит SessionID (строка), которую потом надо подставлять во все запросы в рамках сессии

2) Payment
Нужно вызывать в при внесении денег
в Body передается json в котором передается sessionId,
amount - сумма денег в копейках (то есть для 100 рубей должно прийти 10000, копейки потому что цены у нас тоже в копейках),
inCash - true для оплаты наличкой, false для безнала
В ответ возвращает новый баланс клиента

3) Prices
Возвращает цены на услуги
Название, айдишка усуги (id), Цена в секунду (PricePerSec) и бесплатный интервал использования (FreeDuration)

4) ServiceUsage
Нужно вызывать в при завершении пользования услугой
в Body передается json в котором передается sessionId,
serviceId - айдишка усуги
duration - время использования в секундах
Price - рассчитанная терминалом стоимость использования
В ответ возвращает новый баланс клиента

5) BindSession
Нужно вызывать в при прикладывании карты клиентом, если сессия еще не начата, то надо вызывать StartSession перед этим и потом сразу этот запрос
в Body передается json в котором передается sessionId,
clientId - идентификатор карты клиента
В ответ возвращает баланс клиента

6) StopSession
Нужно вызывать в при завершении сессии - повторное прикладывание карты и\или кончились деньги и\или по времени неиспользования
в Body передается json в котором передается sessionId,
В ответ возвращает баланс клиента
